1. A file to enter the Ips is found in the solution named IpList.txt. Important do not enter duplicate Ips 
	and they should all be seperated by comma

2. Get -> ​/api​/ipdetails​/{ip}

3. Put -> /api
	This will get the ip list from a file (as described in point 1) and Adds or updates the IP in the DB 
	IMP: There is a delay in the BackgroundWorker.cs which will help us to see & test the next call. 
	This timer will eventually be removed

4. Get -> ​/{guid}
	By entering the guid returned in the previous call will give you the status of the whole batch.
